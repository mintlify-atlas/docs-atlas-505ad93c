The SRA Toolkit is a collection of command-line tools from NCBI for working with data in the INSDC Sequence Read Archive (SRA). Use it to download SRA accessions, convert them to FASTQ or SAM/BAM formats, and inspect read data — on local machines or in the cloud.

## Get started quickly

<Steps>
  <Step title="Download and install">
    Get the latest pre-built binaries for your platform from the [download page](https://github.com/ncbi/sra-tools/wiki/01.-Downloading-SRA-Toolkit) or install via Conda.

    ```bash
    conda install -c bioconda sra-tools
    ```
  </Step>
  <Step title="Configure the toolkit">
    Run `vdb-config` to set your cache directory and enable remote access to NCBI data.

    ```bash
    vdb-config --interactive
    ```
  </Step>
  <Step title="Prefetch an accession">
    Download an SRA accession to your local cache with `prefetch`.

    ```bash
    prefetch SRR000001
    ```
  </Step>
  <Step title="Convert to FASTQ">
    Extract reads in FASTQ format using `fasterq-dump`.

    ```bash
    fasterq-dump SRR000001
    ```
  </Step>
</Steps>
